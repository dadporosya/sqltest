Чтобы активировать бота, запустите файл main.py
Можете использовать следующие команды:
/new_project - используй для добавления нового проекта
/skills - чтобы добавить навык к проекту
/projects - чтобы посмотреть проект
/delete - чтобы удалить проект
/update_projects - чтобы обновить проект

В файле logic.py находиться класс, позволяющий оперировать базой данных
